About this Web application

This web application will try to predict whether an employee will stay or leave a company based on data of employee. This application will ask to put some input of employee which have impact on outcome based on historical data.

About the algorithm

Logistic regression is the appropriate regression analysis to conduct when the dependent variable is binary. In logistic regression, analysis is focussed on predicting a state. In this algorithm dependent features are trained to get relation between dependent variables with the variable to be predicted.
